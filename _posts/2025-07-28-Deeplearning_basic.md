머신러닝은 데이터 중심(data-driven) 접근법이다. ( 1. 사람이 모델 형태 설계 2. 모델의 목표 정의 3. 학습데이터를 사용하여 목표를 잘 달성할 수 있는 weight 찾음)
딥러닝이란 - 머신러닝의 방법론 중 하나로써, 생물의 = 인공신경망 = 다층 퍼셉트론(MLP)

퍼셉트론 

활성화함수(Activation Function)

가중합의 값을 판단하는 함수를 화

step function
가중합이 0보다 작으면 0, 0보다 크면 1로 반환 - 미분이 안되어서 Sigmoid를 사용하게됨 => sigmoid를 사용하면서 기울기소실문제 발생

Sigmoid 함수
특징 : 입력값을 0과 1 사이의 값으로 변환해 줍니다.
쓰임새: 주로 이진 분류 문제의 출력층에서 0과 1 사이의 확률값을 얻고자 할 때 사용됩니다.   순환 신경망(RNN)에서도 사용되기도 해요. 
단점: 입력값이 너무 크거나 작으면 기울기가 거의 0이 되어 역전파 과정에서 학습이 잘 이루어지지 않는 '기울기 소실(Vanishing Gradient)' 문제가 발생할 수 있습니다.  또한, 출력값이 0을 중심으로 대칭적이지 않은(not zero-centered) 특성 때문에 학습이 느려질 수 있다는 단점도 있어요. 

하이퍼볼릭 탄젠트(Tanh) 함수
특징: 시그모이드 함수를 변형한 형태로, 입력값을 -1과 1 사이의 값으로 변환해 줍니다. 
쓰임새: 시그모이드와 마찬가지로 순환 신경망(RNN)에서 많이 사용됩니다. 
장점: 시그모이드 함수보다 출력값이 0을 중심으로 대칭적이기 때문에 학습이 더 효율적으로 진행될 수 있습니다. 


ReLU (Rectified Linear Unit) 함수
특징: 입력값이 0보다 작으면 0을 출력하고, 0보다 크면 입력값을 그대로 출력합니다 (max(0, x)).
쓰임새: 심층 신경망(DNN)에서 주로 사용되며, 가장 널리 사용되는 활성화 함수 중 하나입니다. 
장점: 기울기 소실 문제를 완화하고, 계산이 단순하여 학습 속도가 빠르다는 장점이 있습니다.
