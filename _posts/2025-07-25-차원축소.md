## PCA(Principal Component Analysis)- 주성분 분석
차원 축소 기법 중 하나
1. Feature가 너무 많을때
2. 더 적은수의 새로운 특징으로 변환
3. 데이터의 본질적인 정보 손실을 최소화
모델링. 분석 전에 PCA 사용


PCA의 기본 목표 - 데이터의 분산을 가장 잘 설명하는 새로운 축(주성분)을 찾는것

작동방식
1. 데이터의 분산 파악
2. 새로운 축(주성분) 생성
3. 데이터 투영

분산을 최대화하는 것은 잔차 제곱의 최소화
수학적 공간에서 데이터의 분산을 가장 잘 설명하는 고유벡터를 찾는것


공분산(Covariance) : **두 변수**가 함께 **어떻게 변하는가**

양의 공분산 : 두변수가 같은방향으로 움직이는 경향이 크다
음의 공분산 : 두 변수가 반대방향으로 움직이는 경향이 크다
0에 가까운 공분산 : 두 변수 간에 특별한 선형 관계가 없다.

corr과의 차이 - corr(pearson 상관계수) 공분산을 표준편차로 나눈다? 그래서 크기가 -1-1사이의 값을 가짐

공분산식 (수학)
Cov(X,Y) = 1/(n-1) sigma(i=1,n,) (x_i - x_bar)(y_i - y_bar)

bar는 평균ii

공분산 수식(데이터)
C = 1/(N-1) X.T @ X


PCA와 벡터의 내적 

원점 - datapoint = D3 : initial variance
원점 - projected = D1 : remaining variance
datapoint - projected = D2 : lost variance

D3^2 = D1^2 + D2^2
      maximize D1, minimize D2

PCA 작동방식
1. 데이터 표준화
2. 공분산 행렬계산
3. 고유값 및 고유 벡터 계산
4. 주성분 선택 - 고유값이 가장 큰 k개의 고유백터 선택
5. 데이터 변환( 투영) - 저차원 공간으로 변환
